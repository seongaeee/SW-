```
달팽이와 같은 모양으로 배열을 쓰거나 탐색할 때 쓰인다.
```

#### 코드
 ```
int[][] map=new int[5][5];
int max = 5; //배열 크기
int c = -1;
int r = 0;
int d = 1;
int cnt = 0;

while(max>=0){

   for(int i=0; i < max; i++){ //좌or우
      c = c+d;
      map[r][c] = ++cnt;
   }
   --max;

   for(int i=0; i < max; i++){ //상or하
     r = r+d;
     map[r][c] = ++cnt;
   }

   d *= -1; //방향전환
}
 ```
